Dropzone.js

Inputmask

<input class="form-control" id="inputmask" type="text" placeholder="$0.00" data-inputmask="'alias': 'currency', 'numericInput': 'true', 'prefix': '$'" />

<script>

</script>

Tiptap

Tiptap is a headless editor framework with an open-source core. We've created a custom Tiptap class that enhances the default Editor class with several pre-installed extensions and a formatting toolbar. Please refer to the official plugin documentation for detailed information.

<div class="form-control" id="tiptapExample" data-placeholder="Type here..."></div>

<script>
  const editor = new Tiptap({ element: document.getElementById('tiptapExample') });
  console.log(editor.getJSON());
</script>
